ModularExpressions

Source-Generator Error MODEX7

The value of property 'property' in class 'class' contains a call to 'Modex method' with the argument 'argument' which is not a compile-time constant so it cannot be used as part of the calculation of a GenerateModex attribute

Arguments to Modex methods must be compile-time constants (i.e. literal values or const) in order to participate in the calculation of a modular expression.

Example

The following sample generates MODEX7:

// MODEX7.cs
using static ModularExpressions.Modex;

namespace MyNamespace;

public partial class MyClass
{
    [GenerateModex(nameof(DigitsModex))]
    public static partial string DigitsPattern();   // MODEX7
    //                           ^^^^^^^^^^^^^

    private static Modex DigitsModex => Digit.Times(NumberOfDigits);

    public static int NumberOfDigits { get; set; }
}
