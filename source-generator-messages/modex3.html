ModularExpressions

Source-Generator Error MODEX3

"name" does not exist as the name of a property in class 'class' so it cannot be used as part of the calculation of a GenerateModex attribute

The argument provided to the GenerateModex attribute must be the name of a property defined in the class containing the method this attribute decorates.
For discoverability, it is recommended to use the nameof expression.

Example

The following sample generates MODEX3:

// MODEX3.cs
using static ModularExpressions.Modex;

namespace MyNamespace;

public partial class MyClass
{
    [GenerateModex("NoSuchPropertyInThisClass")]
    public static partial string MyPattern();   // MODEX3
    //                           ^^^^^^^^^
}
