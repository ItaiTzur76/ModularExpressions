ModularExpressions

Source-Generator Error MODEX6

A Modex-property structure must be a DAG, but the following property cycle was found under class 'class': Modex-property-cycle

The value of a Modex property can refer to other Modex properties (in the same class) which can, it turn, reference others, too, and so on.
But in order to be resolved into a (finite) string, there should be no cycles in this graph of references.

Example

The following sample generates MODEX6:

// MODEX6.cs
using static ModularExpressions.Modex;

namespace MyNamespace;

public partial class MyClass
{
    [GenerateModex(nameof(Modex1))]
    public static partial string MyPattern();   // MODEX6
    //                           ^^^^^^^^^

    private static Modex Modex1 => Modex2.Times(3);

    private static Modex Modex2 => ZeroOrOneOf(Modex1);
}
