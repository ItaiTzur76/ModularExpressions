ModularExpressions

Source-Generator Error MODEX2

GenerateModex attribute can only be placed on parameterless methods, but method 'method' receives number_of_parameters parameter/s

The suggested code-fix is to remove all parameters.

Example

The following sample generates MODEX2:

// MODEX2.cs
using static ModularExpressions.Modex;

namespace MyNamespace;

public partial class MyClass
{
    [GenerateModex(nameof(AtLeastThreeDigitsModex))]
    public static partial string AtLeastThreeDigitsPattern(int max);   // MODEX2
    //                           ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

    private static Modex AtLeastThreeDigitsModex => Digit.TimesAtLeast(3);
}
