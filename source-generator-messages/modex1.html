ModularExpressions

Source-Generator Error MODEX1

GenerateModex attribute can only be placed on string-returning methods, but method 'method' returns 'type'

The suggested code-fix is to change the return type to string.

Example

The following sample generates MODEX1:

// MODEX1.cs
using static ModularExpressions.Modex;

namespace MyNamespace;

public partial class MyClass
{
    [GenerateModex(nameof(ThreeDigitsModex))]
    public static partial int ThreeDigitsPattern();   // MODEX1
    //                    ^^^

    private static Modex ThreeDigitsModex => Digit.Times(3);
}
