ModularExpressions

Source-Generator Error MODEX5

'member' in class 'class' is not a property so it cannot be used as part of the calculation of a GenerateModex attribute

Only members defined as properties can participate in the calculation of a modular expression; other members (such as methods) cannot, even if they have a Modex return type.

Example

The following sample generates MODEX5:

// MODEX5.cs
using static ModularExpressions.Modex;

namespace MyNamespace;

public partial class MyClass
{
    [GenerateModex(nameof(FiveCapitalVowelsModex))]
    public static partial string FiveCapitalVowelsPattern();   // MODEX5
    //                           ^^^^^^^^^^^^^^^^^^^^^^^^

    private static Modex FiveCapitalVowelsModex => CapitalVowelModex().Times(5);

    private static Modex CapitalVowelModex() => AnyCharacterIn('A', 'E', 'I', 'O', 'U');
}
